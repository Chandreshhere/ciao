Come closer, and listen.

The hand to the right is an iframe with a transparent background. It sits over this page's gradient and content — no white box, no chrome. Use the controls below to swap its pose live.

See the embed code Full version with all tweaks

Live controls

Click a pose. We send a postMessage to the iframe — no reload, no re-fetch of the 3D model.

Paste into your site

Drop this iframe wherever you want the hand. Size it however you like — the canvas is responsive.

<!-- transparent widget, default come-here loop -->
<iframe
  src="https://your-site.com/hand-widget.html?pose=beckon"
  style="width:100%; height:560px; border:0; background:transparent;"
  allowtransparency="true"
  loading="lazy">
</iframe>

// Change pose at runtime from your page:
document.querySelector('iframe').contentWindow.postMessage(
  { type: 'pose', value: 'grab' }, '*'
);
